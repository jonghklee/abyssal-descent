# Abyssal Descent

A roguelike dungeon crawler with deep progression, gacha mechanics, and infinite replayability.

## Play
Open `index.html` in any modern browser.

## Controls
- **WASD** — Move
- **Mouse** — Aim
- **Click** — Attack (hold for charge attack)
- **Space** — Dash (invincible frames)
- **Q** — Potion
- **E** — Switch weapon
- **R** — Weapon Forge
- **T** — Slot Machine
- **F** — Combo Finisher (when ready)
- **C** — Codex
- **M** — Minimap toggle
- **Tab** — Stats panel
- **Esc** — Pause
- **1-4** — Active skills

## Features
- 4 Classes (Warrior, Rogue, Mage, hidden Necromancer)
- 20 floors + endless mode with Ascension system
- 5 biomes with unique visuals and music
- 20 relics with synergy combos
- 13 collectible pets with evolution
- Gacha system, slot machine, lucky wheel
- 35 achievements, Codex collection
- Save/Load, meta-progression (Soul Forge)
- Daily challenges

## Tech
- Pure HTML5 Canvas + vanilla JavaScript
- Web Audio API for sound
- localStorage for saves
- ~12,600 lines of code
